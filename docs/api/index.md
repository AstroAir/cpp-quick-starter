# API Reference / API 参考

<!-- [EN] -->
The API documentation is generated by Doxygen.

To generate the documentation:
<!-- [/EN] -->

<!-- [ZH] -->
本项目 API 文档由 Doxygen 生成。

生成文档的方法：
<!-- [/ZH] -->

```bash
cmake --preset ninja-debug -DCPP_QUICK_STARTER_BUILD_DOCS=ON
cmake --build --preset ninja-debug --target doxygen
```

<!-- [EN] -->
After generation, access the HTML documentation at:
<!-- [/EN] -->

<!-- [ZH] -->
生成后可通过以下路径访问 HTML：
<!-- [/ZH] -->

`build/ninja-debug/docs/doxygen/html/index.html`
